# Pomodoro Timer

A simple Pomodoro timer web app for the 100-days-of-code project.

Features:
- Start / Pause / Reset timer
- Adjustable work and break durations
- Optional auto-switch between work and break
- Persist settings with localStorage

How to use:
1. Open `index.html` in your browser
2. Adjust the work and break durations (minutes)
3. Click Start to begin
4. Enable "Auto-switch" to switch to break mode automatically when timer ends

You can customize the sound or styles as you like.
